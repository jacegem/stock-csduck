# 주식으로 용돈벌기

## 사전 준비
 `대신증권` 증권계좌를 이용하여 주식을 합니다. 따라서 사전에 계좌 개설이 되어 있어야 합니다. 
 
 CybosPlus 가 설치되어 있어야 합니다. 인증서 로그인을 위한 증권거래용 인증서가 필요합니다.
